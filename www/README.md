# Reactor
## Worb Corp.'s starter kit for React front-end projects.

### Build
Assumes nodenv with the version in .node-version installed.

1. `npm install --global webpack`
2. `npm install --global webpack-dev-server`
3. `npm install`
4. `webpack --watch`
